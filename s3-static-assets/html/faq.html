💡 How do I clear my multi-node session state?
Simply click the red "Logout" button on your main dashboard. This instructs the controller to instantly purge your session key across the distributed Redis memory layer.
🔒 Is my connection to the infrastructure secure?
Yes! All node authentications utilize decentralized session validation hashes, isolated from the local file system.
